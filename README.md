# BGM Data  

This project automates the process of converting an Excel sheet of blood glucose readings formatted for easy data entry into a CSV file formatted for data visualization in Tableau. 
  
**Blood Glucose Readings.ipynb**  
Python code for reading data from an Excel spreadsheet, validating the results, and creating a CSV file to use with Tableau.

**Blood Glucose Readings Sample.xlsx**  
Excel spreadsheet with blood glucose readings from a blood glucose monitor (BGM).

**Guide to Blood Glucose Monitor Visualizations.pdf**  
A reference guide describing the visualization styles, analytical approaches, and commonly used BGM measures.
